# eslint-config-patrio
A set of ESLint (http://eslint.org) opinionated rules by me.

## Usage:
1. `npm install --save-dev eslint-config-patrio babel-eslint eslint-plugin-react`
2. Create a file named `.eslintrc` in your project:
```js
{
  "extends": "patrio"
  // Your overrides...
}
```
